Webページで再生中の音楽をリアルタイムでアナライズします。

-40dbから0dbまでのピークメーター、そして位相モニターをサポートします。

リサージュ波形とCORRモニターを利用し、位相のずれやLRの片寄りを視覚的に把握できます。

Webの音声がおかしいと思ったとき、また、配信などで自分の音声が正常なのかを客観的なメーターで把握でき、音声のエラーが自分の環境に依存するのかソースに依存するのかを切り分けられます。
